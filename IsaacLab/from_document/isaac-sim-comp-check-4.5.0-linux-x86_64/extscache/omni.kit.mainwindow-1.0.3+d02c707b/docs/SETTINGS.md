# Settings

## Settings Provided by the Extension

### `ext."omni.kit.mainwindow".margin.width`
- **Default Value:** 0
- **Description:** Sets the width of the margin around the main application window.

### `ext."omni.kit.mainwindow".margin.height`
- **Default Value:** 0
- **Description:** Sets the height of the margin around the main application window.

### `ext."omni.kit.mainwindow".backgroundColor`
- **Default Value:** 0xFF1F2124
- **Description:** Specifies the background color of the main application window.

### `ext."omni.kit.mainwindow".startup.dockWindows`
- **Default Value:** true
- **Description:** Determines whether to dock windows on application startup.
